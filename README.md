# World Atlas

World Atlas is a modern and responsive web application built with React and Tailwind CSS.  
It allows users to explore countries around the world, search dynamically, filter by region, and view detailed country information through a clean and intuitive interface.

---

## Overview

This project was built to practice modern React development patterns including dynamic routing, reusable components, state management, and API integration. The application features a consistent dark-themed UI designed using Tailwind CSS.

---

## Features

- Browse and explore all countries
- Search countries in real time
- Filter countries by region
- View detailed country information
- Dynamic routing using React Router
- Responsive layout for all screen sizes
- Custom error handling page

---

## Tech Stack

- React
- React Router v6
- Tailwind CSS
- REST Countries API
- Vite

---

## Project Structure

The application follows a clean component-based structure:

- Layout components for shared UI (Header, Footer, AppLayout)
- Reusable UI components (Loader, SearchFilter, CountryCard)
- Separate pages for each route
- Centralized routing configuration
- Tailwind-based styling system
